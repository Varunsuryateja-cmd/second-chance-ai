# 🚀 Second Chance AI

> Don't Just Remember. Finish.

## 📌 Problem Statement
**The Last-Minute Life Saver**

Students, professionals, and entrepreneurs often miss deadlines, assignments, meetings, and important commitments because traditional reminder apps only send notifications and do not actively help users complete tasks.

---

## 💡 Solution Overview

Second Chance AI is an AI-powered productivity companion that proactively helps users organize tasks, prioritize work, monitor deadlines, and improve productivity through intelligent recommendations and visual insights.

Instead of simply reminding users, the application helps them take meaningful action before deadlines are missed.

---

## ✨ Key Features

### 📝 Task Management
- Add and manage tasks with deadlines
- Set estimated completion hours
- Assign task priorities (High, Medium, Low)
- View all tasks in one place

### 🧠 Intelligent Task Prioritization
- Automatically identifies urgent tasks
- Highlights at-risk tasks
- Provides smart task ordering

### 🤖 AI-Powered Scheduling Assistance
- Generates personalized plans
- Suggests focus sessions
- Helps users allocate time effectively

### 📊 Productivity Dashboard
- Completion probability
- At-risk task counter
- Focus time availability
- Upcoming tasks section
- Energy forecast visualization

### 📅 Calendar Integration
- Interactive calendar view
- Displays current date
- Helps users plan upcoming tasks

### 📈 Productivity Insights
- Completion statistics
- Productivity score
- Completion trend charts
- Weekly progress tracking

### 🎯 Goal and Habit Tracking
- Productivity metrics
- Task completion tracking
- Daily progress monitoring

### 🎤 Voice-Enabled Assistance
- Voice input support
- Hands-free interaction
- Faster task entry

### 🌙 Modern User Experience
- Dark Mode and Light Mode
- Responsive dashboard interface
- Modern glassmorphism-inspired design

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Varunsuryateja-cmd/second-chance-ai.git
cd second-chance-ai
```

### Backend Setup

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```

Backend URL:

```
http://127.0.0.1:8000
```

Swagger Documentation:

```
http://127.0.0.1:8000/docs
```

### Frontend Setup

Open:

```
frontend/login.html
```

or run using VS Code Live Server.
